# 👕 Casual T-Shirt Product Page

## 📌 About

Casual T-Shirt Product Page built with a clean and responsive design.  
This project showcases a modern product layout including product images, description, pricing, sizes, quantity selection, and Buy Now button functionality.  
Perfect for practicing UI/UX design and front-end development concepts.

## 🚀 Features

- 🖼️ Product Images (Multiple Views)
- 📝 Product Description
- 💲 Price Display
- 📏 Size Selection
- 🔢 Quantity Selector
- 🛒 Buy Now Button
- 📱 Fully Responsive Design

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & Layout
- **JavaScript (JS)** – Interactivity
